## Automata Graph Validator #### Proyecto Final Análisis de Algoritmos 2015-2 ![Logo USTA](./images/algoritmos-usta-2015/logo.jpg "Logo USTA") Integrantes: * Seidler, Ezequiel Nicolás * Cravacuore, Damián
### ¿Qué herramientas utilizamos?
Flask Python Github Heroku TravisCI AngularJS Foundation
### Python 2.7 ![python](./images/algoritmos-usta-2015/python-logo.png) * Back-end * OOP * Ligero * Rápida configuración * Relacionado a framework..
### Flask ![flask](./images/algoritmos-usta-2015/flask-logo.png) * Micro Framework para Python * Jinja2 Templating * Buena documentación * Simple y rápido
### AngularJS ![angular](./images/algoritmos-usta-2015/angularjs-logo.png) * Facilidades para front-end * Directivas ng-repeat, etc. * Framework popular * Dynamic single-page apps
### ZURB Foundation ![foundation](./images/algoritmos-usta-2015/foundation-logo.png) * No es Bootstrap :P * Responsive design * Componentes AngularJS con angular-foundation
### Github ![github](./images/algoritmos-usta-2015/github-logo.png) * Repositorio git * Buena disponibilidad * Integración con gran variedad de servicios
### TravisCI ![travis](./images/algoritmos-usta-2015/travisci-logo.png) * Servidor de Integración Continua * Simple * Integrado con Github * Automatización de tareas (Tests!)
### Heroku ![heroku](./images/algoritmos-usta-2015/heroku-logo.png) * Deploy server * Cloud platform * Gratuito para pequeños proyectos * Simple integración con Github
## ¿Qué metodologías empleamos?
### TDD - Test-Driven Development * Diseño mientras creo mis tests * Modelo acotado, sin cosas de más * Aseguro que no rompo funcionalidades anteriores con nuevos cambios ![tdd](./images/algoritmos-usta-2015/tdd.png)
### Mockups * Mostrar ideas de diseño al 'cliente' * Prototipado antes de programar * Estructura la navegación ![mockup1](./images/algoritmos-usta-2015/mockup-table.png) ![mockup2](./images/algoritmos-usta-2015/mockup-validation.png)
### UML - Unified Modeling Language * Diseño de ideas * Modelado antes de programar ![uml](./images/algoritmos-usta-2015/uml.png) ![uml_final](./images/algoritmos-usta-2015/uml_final.png)
