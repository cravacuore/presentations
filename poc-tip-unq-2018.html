## Presentación Prueba de Concepto TIP #### rawRSS - TIP - UNQ - 2018 ![Logo rawRSS](./images/poc-tip-unq-2018/rawRSS-icon.png "Logo rawRSS") Cravacuore, Damián
### Motivación * Qué es RSS? #reviveRSS * Rich Site Summary * Standard (Wordpress) * Lector favorito (Inoreader) * Privativo * Features innecesarios * Integración con redes sociales * otros features pagos * No es local, no puedo leer offline
![inoreader](./images/poc-tip-unq-2018/inoreader-example.png)
### Alcance * Manejo de feeds * Posibles integraciones: * Canales YouTube * Spotify (band follow) * Ideas futuras * 'Inteligencia' sobre actualización de feeds (AI) * Deploy 'in the cloud' * Python backend sobre AWS
